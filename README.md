# p0-w3-d2-t3 · Функции-форматтеры

> Тип: написание функций · Репозиторий: один отдельный

> Подключить файл, отсмотреть в браузере.

## Задача

В любом интерфейсе данные нужно приводить к красивому виду: цену — с разделителями и валютой, имя — с заглавной буквы, приветствие — собрать из кусочков. Это классические маленькие функции, которые принимают данные и возвращают готовую строку. Пишешь набор таких и переиспользуешь.

```
функции-форматтеры: приняли данные → вернули строку
  formatPrice(1990)   → '1 990 ₽'
  formatName('аня')   → 'Аня'
  greet('Аня', 'утро')→ 'Доброе утро, Аня!'
вызываешь с разными аргументами, переиспользуешь
```

## Что нужно сделать

В `format.js` написать функции-форматтеры и показать их работу на нескольких входах: `formatPrice(number)` → строка с разделением разрядов и знаком валюты; `formatName(string)` → имя с заглавной первой буквы, остальные строчные; `greet(name, partOfDay)` → собранное приветствие. Вызвать каждую минимум на 3 разных аргументах.

## Критерии приёмки

* Каждая функция принимает аргумент(ы) и возвращает строку (а не печатает внутри себя).
* Форматтеры вызваны на нескольких разных входах, результат выводится при вызове.
* `formatName` корректно работает и на строчном, и на уже-заглавном вводе; `greet` собирает строку из переданных кусков.
